Installation

Choose the appropriate embed code based on your needs:

Responsive Embed (Recommended)

This version automatically adapts to both desktop and mobile:

<script src="https://your-domain.com/embed.js" async></script>

Desktop-Optimized Version

Use this if you want to show the calculator only on desktop devices:

<script>
if (window.innerWidth >= 768) {
  const script = document.createElement('script');
  script.src = 'https://your-domain.com/embed.js';
  script.async = true;
  document.body.appendChild(script);
}
</script>

Mobile-Optimized Version

Use this if you want to show the calculator only on mobile devices:

<script>
if (window.innerWidth < 768) {
  const script = document.createElement('script');
  script.src = 'https://your-domain.com/embed.js';
  script.async = true;
  document.body.appendChild(script);
}
</script>
